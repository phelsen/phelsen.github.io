f.eq(value1,value2)
Testing on equality. Does a deep comparisation on both type and content. Also compares empty collections 
f.eq(1,1)   // => true 
f.eq(1,"1")   // => false 
f.eq([1,2,3],[1,2,3]) //  // => true 
f.eq ([],[])  // => true 
f.eq({},[])   // => false 
f.eq({a : [1, 2,3], b : [1, 2, [3, 4, { c : 42}]]},
     {a : [1, 2,3], b : [1, 2, [3, 4, { c : 43}]]})  // => false
Since keys in maps/js objects don't guarantee order, equality does not take key order into account. 
f.eq( { a : [1,2], b : 99}, { b: 99, a: [1,2]})    // => true
f.clone(value)
Returns a deep clone. 
const a = [1,2, { x : 42, y: 43}]
const b = f.clone(a)
a[2].y = 99
a // =>  [ 1, 2, { a: 42, b: 99 } ]
b // => [ 1, 2, { a: 42, b: 43 } ]]
f.first(coll)
Returns the first element of a collection. This is the first element of the array, the first character of a string or the first key/value-pair of a map. 
f.first([1,2,3,4])  // =>  1 
f.first("BarFoo")  // =>  "B"  
f.first ({ a : 12, b : 42, c : 99  })     // =>  ["a",12]
f.second(coll)
Returns the second element of a collection. This is the second element of the array, the second character of a string or the second key/value-pair of a map. 
f.second([1,2,3,4])  // =>  2 
f.second("BarFoo")  // =>  "a"  
f.second ({ a : 12, b : 42 , c : 99})     // =>  ["b",42]
f.last(coll)
Returns the last element of a collection. This is the last element of the array, the last character of a string or the last key/value-pair of a map. 
f.last([1,2,3,4])  // =>  4 
f.last("BarFoo")  // =>  "o"  
f.last ({ a : 12, b : 42 , c : 99  })     // =>  ["c",99]
