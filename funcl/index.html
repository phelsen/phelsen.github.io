Funcl

f.eq(value1,value2)
Testing on equality. Does a deep comparisation on both type and content. Also compares empty collections 
f.eq(1,1)   // => true 
f.eq(1,"1")   // => false 
f.eq([1,2,3],[1,2,3]) //  // => true 
f.eq ([],[])  // => true 
f.eq({},[])   // => false 
f.eq({a : [1, 2,3], b : [1, 2, [3, 4, { c : 42}]]},
     {a : [1, 2,3], b : [1, 2, [3, 4, { c : 43}]]})  // => false
Since keys in maps/js objects don't garantee order, equality does not take key order into account. 
f.eq( { a : [1,2], b : 99}, { b: 99, a: [1,2]})    // => true
f.clone(value)
Returns a deep clone. 
const a = [1,2, { x : 42, y: 43}]
const b = clone(a)
a[2].y = 99
a // =>  [ 1, 2, { a: 42, b: 99 } ]
b // => [ 1, 2, { a: 42, b: 43 } ]]
